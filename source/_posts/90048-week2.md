## Term
在Prolog中，所有的数据结构都被叫为Term（中文该翻译成什么我也不知道...），Term可以被分为：
* 单个的（Atomic）：包括整数和浮点数，小写字母开头
    * 例如：``a``，``queen_elizabeth``，``banana``
* 复合的（Compound）：有一个算符，类似函数名，后面可能有参数跟着，逗号分开
    * 例如：``node(leaf, 1, node(leaf, 2, leaf))``
* Ground Term：没有变量的Term
    * 例如：``3``，``f(a, b)``
* Nonground Term：拥有至少一个变量的Term
    * 例如：``Name``，``f(a, X)``

## 变量（Variable）
变量也是Term，以一个大写字母或下划线开头。
单个的下划线比较特殊，它被称为匿名变量。

## 列表（List）
在Prolog中，一个空的列表用``[]``表示。
``[1, 2, 3]``表示一个拥有1、2、3这三个元素的列表。
``x:xs``表示一个列表以x为头，xs为尾巴。

### 识别列表
```prolog
proper_list([]).
proper_list([Head|Tail]) :-
    proper_list(Tail).
```

### 添加操作
```prolog
append([], C, C).
append([A|B], C, [A|BC]) :-
    append(B, C, BC).
```

### 长度操作
```prolog
length([a, b, c], Len).
Len = 3.
```
```prolog
length(List, 3).
List = [_2956, _2962, _2968].
```

## 替换（Substitution）
替换是一个将变量转换为Term的过程，当它被执行时，所有的变量都会被替换成Term。
* 例如：对``node(X1, X2, X3)``执行``{X1->leaf, X2->1, X3->leaf}``会得到``node(leaf, 1, leaf)``

## 联合（Unificaton）
联合是将两个Term变得相同，通过绑定变量，以下成立：
* 如果一个Term是变量，被绑定另一个Term
* 如果两个Term是相同的单个的Term
* 如果两个Term是复合的，拥有同样的算符，参数的配对可以被联合

## 运算
错误的方式：
```prolog
X = 6 * 7.
X = 6*7.
```
正确的方式：
```prolog
X is 6 * 7.
X = 42.
```

* ``+``,``-``,``*``：加、减、乘
* ``/``：除法（可以返回一个浮点数）
* ``//``：整数除法
* ``mod``：求余
* ``<``,``=<``：小于，小于等于
* ``>``,``>=``：大于，大于等于
* ``=:=``,``=\=``：等于，不等于（只针对数字）
