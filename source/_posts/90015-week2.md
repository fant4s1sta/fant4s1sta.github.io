## 端口（Port）
* TCP和UDP协议都是使用端口来进行不同应用对接收数据的匹配处理。
* 端口用一个16bit的正整数表示：
    * ftp：21/tcp
    * telnet：23/tcp
    * smtp：25/tcp
    * http：80/tcp
    * login：513/tcp
* 用户层面通常使用的端口都是大于1024的。

## 套接字（Socket）
* 套接字提供了运输层里网络编程的接口。
* 使用套接字的网络通讯与文件I/O非常相似。
* 套接字通讯不需要相同的编程语言。

## 套接字通讯
1. 我们需要一个Server在一台计算机运行并且拥有一个绑定到特殊端口的套接字。服务器需要等待和听取Client连接请求的套接字。
2. 如果一切正常，Server接收连接请求。同时，因为端口被用于连接，Server会需要一个绑定新端口的新套接字，通过它可以在连接Client时听取原来套接字的连接请求。

## Java套接字
Java的.net包中提供了两个类：
* Socket：用于实现Client
* ServerSocket：用于实现Server

### 实现Server
1. 打开Server的套接字
```java
ServerSocket server;
DataOutputStream os;
DataInputStream is;
server = new ServerSocket(PORT);
```
2. 等待Client的请求
```java
Socket client  = server.accept();
```
3. 创建I/O流来与Client通讯
```java
is = new DataInputStream(client.getInputStream());
os = new DataInputStream(client.getOutputStream());
```
4. 进行与Client的通讯
```java
String line = is.readLine();
os.writeBytes("Hello\n");
```
5. 关闭套接字
```java
client.close();
```

**如果是多线程的Server**
```java
while (true) {
    // 首先，执行步骤1和步骤2
    // 创建一个线程，用Client套接字作为参数，这个线程会创建I/O，然后通讯
}
```

### 实现Client
1. 创建一个套接字对象
```java
client = new Socket(server, port_id);
```
2. 创建I/O流来与Server通讯
```java
is = new DataInputStream(client.getInputStream());
os = new DataOutputStream(client.getOutputStream());
```
3. 进行与Server的通讯
```java
String line = is.readLine();
os.writeBytes("Hello\n");
```
4. 关闭套接字
```java
client.close();
```
