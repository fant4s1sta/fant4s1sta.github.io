## 关系（Relations）
在Prolog中表示一种亲子关系：
```prolog
parent(queen_elizabeth, prince_charles).
```
这里有两个参数，第一个代表Parent，第二个代表Children。
同时，这个关系叫作叙述（predicate）。

## 事实（Facts）
一个这样的声明被叫作事实：
```prolog
parent(queen_elizabeth, prince_charles).
```

## 查询（Queries）
在Prolog中，我们可以执行一些查询：
```prolog
parent(prince_charles, prince_william).
```
该查询执行结果为true。

```prolog
parent(prince_willian, prince_charles).
```
该查询执行结果为false。

## 变量（Variable）
变量为叙述里的一个参数，在Prolog中以一个大写字母或下划线开头：
```prolog
parent(prince_charles, X).
```
该查询结果为：
```prolog
X = prince_william;
X = prince_harry.
```
Prolog会返回查询的所有正确结果。
同时，也可以代入多个变量进行查询：
```prolog
parent(X, Y).
```
该查询结果为：
```prolog
X = queen_elizabeth,
Y = prince_charles;
X = prince_philip,
Y = prince_charles;
.
.
.
```

## 复合查询（Compound Queries）
查询也可以使用多个叙述语句，逗号可以被看成”and“：
```prolog
parent(queen_elizabeth, X), parent(X, Y).
```
该查询会返回满足所有叙述的结果：
```prolog
X = prince_charles,
Y = prince_william;
X = prince_charles,
Y = prince_harry.
```

## 规则（Rules）
我们可以通过规则来定义叙述，规则是这样的形式：
```prolog
Head :- Body,
```
这里Head以一个事实的形式，而Body以查询的形式：
```prolog
grandparent(X, Z) :- parent(X, Y), parent(Y, Z).
```
这里的意思为“X是Z的祖父如果X是Y的父亲并且Y是Z的父亲”。

## 递归（Recursion）
很多时候规则可以用递归来表示，在Prolog中我们没有循环语句，所以我们用递归：
```prolog
ancestor(Anc, Desc) :-
    parent(Anc, Desc).
ancestor(Anc, Desc) :-
    parent(Parent, Desc),
    ancestor(Anc, Parent).
```

## 相等（Equality）
在Prolog中，我们用“=”来表示相等，也可以进行绑定：
```prolog
a = b.
```
这里返回结果为false。
```prolog
X = 1.
```
这里返回结果为：
```prolog
X = 1.
```

## 逻辑Or（Disjunction）
在Prolog中，我们用“;”来表示逻辑Or：
```Prolog
parent(queen_elizabeth, X) ; parent(princess_diana, X).
```
这里返回结果为满足第一个叙述或第二个叙述的结果。

## 否认（Negation）
在Prolog中，否认被写作“\+”：
```prolog
parent(X, prince_william), \+ X = prince_charles.
```
这里返回的是Price William的父母，但是除了Prince Charle外。

同时，我们也可以使用“\=“，不过有点小不同：
```prolog
parent(X, prince_william), X \= prince_charles.
```
你喜欢哪一个？我喜欢后者～

### 否认失败
Prolog执行\+G时会先试图证明G，如果这个失败，那么结果为true，反之为false。
